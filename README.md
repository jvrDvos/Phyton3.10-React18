## Run Locally

Clone the project

```bash
  git clone https://github.com/jvrDvos/PhytonReact.js
```

Go to the project directory


For the backend
```bash
  start Xampp
  create BD
  open terminal
  cd folder api..
  py run.py


For the frontend

Navagate to the backend and frontend folder, install dependencies and start the server

```bash
  cd front-en...
  open with ide
  npm install
  npm start
```
